# Security Policy

## Scope

Scenario Planning Atlas is a portfolio project that demonstrates strategy
modeling, scenario framing, and executive operating design. It does not process
live customer or production data.

## Reporting

If you identify a security concern, avoid publishing sensitive details in a
public issue. Reach out through:

- [Kinetic Gain](https://kineticgain.com/)

## Safe Defaults

- no secrets are committed
- all data is synthetic
- environment values are documented in `.env.example`
- CI validates the frontend on push and pull request
